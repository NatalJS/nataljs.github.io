NatalJS

NatalJS é um grupo de desenvolvedores - entusiastas, profissionais e iniciantes -
que tem como objetivo comum promover o desenvolvimento de uma comunidade para compartilhamento
de conhecimentos, contribuição com projetos de software livre entre outras atividades.
A comunidade funciona tanto online quanto presencialmente. Os encontros presenciais podem
ter seu local alterado, porém sempre com os mesmos objetivos.

Objetivos

  • Disseminar conehcimentos, novas tecnologias e boas práticas sobre JavaScript
  • Aproximar programadores locais, internacionais e grupos de instituições acadêmicas distintas interessados na mesma área
  • Organizar encontros, conferências e até contribuir com projetos open-source relacionados aos estudos
  • Instigar o movimento comunitário entre os membros para desenvolvimento de projetos e ideias em conjunto.

Como Contribuir

  • Junte-se a nós
  • Venha para o próximo encontro
  • Use a hashtag #nataljs
  • Divulgue-nos
  • Contribua em nossos projetos
